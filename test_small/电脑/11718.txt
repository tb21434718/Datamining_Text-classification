
 
　　当初Netscape 6.0正式版发布真是相当失败，由于存在一些比较严重的问题，因而饱受用户和评论家的批评与挖苦，甚至有人认为其正式版应该等到Mozilla基本成熟时再发布，而不明智地过早推出只能加速Netscape走向没落的步伐。
　　此后作为Netscape 6.0原型的Mozilla则不断更新、升级，可用性也不断提高，到了0.7版本，我们感觉它已经比较完善，能够应付用户基本的使用要求。可能也是由于这个原因，昨天（2月8日），网景公司终于发布了Netscape 6.01，并使用了Gecko引擎的最新版本（Build ID 20010131，而Mozilla 0.7使用的是Build ID 20010109）作为核心。我们对这个6.01版进行了试用，下面请看我们为大家推出的简报。
　　增加安装组件
　　Netscape 6.01完全版的安装文件大约有31MB(点这里立即下载)，程序组件也较以前有了大量增加，增加的可选组件包括：Realplayer 8.0、Flash Player、HP打印机标识、德、法、日语语言包、Stamps.com Internet Postage等。安装完毕打开程序时，我们还注意到Netscape 6.01的启动速度也有较大提高。
　　What's New
　　由于Netscape 6.01使用了Gecko Build ID 20010131，因此在Mozilla 0.7中出现的新变化也可以在它这里找到，当然我们也就没有必要再对它们进行重复，有兴趣的朋友可以查看Mozilla 0.7试用简报。
　　该版本浏览性能较Netscape 6.0提高不少，尤其是中文网页显示方面有了质的飞跃，可以说现在你不用再提心掉胆地看中文网站了。在显示效果方面，除细小地方外，基本能够达到IE 5.X所呈现的样子。繁体中文内码识别速度较前一版快很多。并且网页显示速度也有很大提高，可惜感觉上不如Mozilla 0.7那样迅速。
　　在Netscape界面主题网页上提供有七个新主题，下载应用也非常方便、快捷，只需点击网页上的“导入主题”按钮，浏览器即可自动下载并应用。
　　在Siderbar书签项的右键菜单中增加了新的选项：设置成新的Personal Toolbar文件夹、设置成新的Internet搜索文件夹、设置成新的书签文件夹。当我们将某个书签文件夹设置成Personal Toolbar文件夹后，程序锁死，并且再次启动后，设置的文件夹并没有出现在Personal Toolbar工具栏上。
　　搜索功能得到很大改进，现在可选择同时查询十三个搜索引擎，而且还提供更为详细的分类选择。用户还可以对搜索功能进行编辑，比如：增加分类、选择分类使用的搜索引擎等。
　　存在的不足
　　首先令我们奇怪的是在Mozilla 0.7中地址栏增加了右键命令，而Netscape 6.01没有，这样就无法对地址进行复制、剪切、粘贴等快捷操作；在中文输入方面仍然存在以前版本的问题，比如：地址栏输入中文后切换至英文字母会以相反顺序显示、E-Mail组件不支持万能五笔输入法等；应用除标准以外的界面主题后程序有时会出现界面花屏现象；无法设置代理服务器等。
　　总结
　　在试用Netscape 6.01后，我们认为它与6.0版本相比各方面都有了提高，尤其是中文网页显示更是产生了质的飞跃，当然这些进步主要还是得益于Mozilla的不断改进与升级。另外，我们感觉它的稳定性表示还算不错，在试用过程中虽然遇到程序锁死的现象，但并不常见，应该说属于可以接受的正常范围。所以Netscape 6.01尽管还不能称得上是一款非常优秀的浏览器，但它基本能够满足用户一般浏览要求，如果你对Netscape曾经有着一份深厚的感情，那么相信这个版本不会让你再一次失望。
