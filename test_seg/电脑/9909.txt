新加坡 资讯 通信 发展 管理局 和 升阳 （ Sun ） 合作 ， 在 我国 设立 Java 智能 服务 实验室 和 Java 无线 科技 研究 中心 。 
 
 　 　 双方 自 1998 年 建立 Java 科技 合作 计划 ， 至今 已经 投资 2500 万元 ， 建立 多个 Java 研究 中心 。 四年 来 ， 合作 计划 取得 明显 成绩 ， 在 计划 下 成立 的 公司 开发 出 许多 应用 与 服务 ， 其中 许多 已经 推向 海外 市场 。 
 
 　 　 新 成立 的 Java 智能 服务 实验室 将 重点 进行 网络服务 方面 的 研究 及 应用软件 开发 ， 它 的 目标 是 把 我国 建成 本 区域开发 网络服务 的 枢纽 。 Java 无线 科技 研究 中心 则 根据 资信 局 与 升阳 去年 签署 的 协议 而 成立 。 
 
 　 　 Java 是 由 升阳 发明 的 一种 软件 编写 语言 ， 其 特点 是 不受 电脑 平台 限制 ， 可 在 任何 电脑 或 其它 无线 或 有线 电器 上 使用 。 目前 ， 它 已经 成为 编写 电脑 程序 的 主要 语言 ， 全世界 大专 学府 都 教授 这个 语言 。 
 
 　 　 至今 ， 本地 已经 有 超过 100 家 公司 加入 计划 ， 开发 出 的 应用 与 服务 超过 50 项 。 资信 局局长 曾庆怡 、 升阳 执行 副 总裁兼 首席 策略 官 马克 . 托利 佛 （ Mark   Tolliver ） 昨天 出席 了 两个 新 Java 研究 中心 的 成立 仪式 。 
 
 　 　 新 中心 投资 超过 800 万元 ， 它 将 集中 进行 网络服务 的 开发 工作 。 它 将 与 相关 的 政府 与 私人机构 合作 ， 将 相关 服务 通过 互联网 集合 在 一起 ， 为 用户 提供方便 。 
 
 　 　 中心 的 另外 一个 重要 任务 就是 进行 科技 转移 ， 把 升阳 最新 科技 转移 给 有关 公司 和 研究 机构 。 培训 Java 研究 人员 也 是 中心 的 一项 重要 工作 。 
 
 　 　 至于 Java 无线 科技 研究 中心 ， 它 的 任务 是 帮助 我国 成为 亚太地区 开发 无线 应用 与 服务 的 枢纽 。 它 将 为 本地 公司 提供 研究 设备 ， 为 它们 提供 咨询服务 ， 帮助 它们 在 科技 上 先行一步 。 
 
 　 　 它 目前 提供 的 服务 包括 ： 为 开发商 提供 无线 Java 论坛 ； 为 开发商 和 合作伙伴 提供 下载 Java 无线 应用软件 的 服务 。
